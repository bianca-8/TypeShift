### Description
TypeShift is a spin on a cypher. If you type with your hands shifted a bit on a keyboard, you know where the letters are (if you're a good enough typer...), you might be typing gibberish. But with this, your friends can decipher your messages! You can also encode your messages with letters shifted different amounts. 

### Features
- Shift the keys by different amounts (if it goes to the end, it wraps over. For example, if you want to shift by -1 for Q, it goes to P.)
- You can copy the text on both the input and outputs. 
- Switch between decryption and encryption by clicking the button under the boxes, which switches from shifting (+) to (-).
